Minimal Callout with no title and default variant:

```jsx
<Callout>This is a minimal callout</Callout>
```

Examples of each supported variant (with a title):

```jsx

    <p>This is the text of a Callout for the <strong>primary</strong> variant.</p>
    <p>It can contain pretty much any content inside.</p>
</Callout>
```
